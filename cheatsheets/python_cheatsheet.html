Escape Sequence

An escape sequence is a sequence of character that does not represent itself. Instead it is translated into another character when used inside a string literal or character. Escape sequence are as follows: 


print("\\") # Backslash escape sequence
print("\'") # Single quote escape sequence
print("\"") # Double quote escape sequence
print("\n") # Newline escape sequence
print("\t") # Tab escape sequence
print("\r") # Carriage return escape sequence
print("\b") # Backspace escape sequence
print("\f") # Form feed escape sequence
print("\v") # Vertical tab escape sequence
print("\a") # Bell/alert sound escape sequence
print("\ooo") # Octal value (e.g. "\141" is 'a')
print("\xhh") # Hex value (e.g. "\x61" is 'a')

Operators

Operators are the character that is used to perform arithmatic, logical or comarision operation on the values or objects.

Loops in Python

A loop or iteration statement repeatedly execute a give statement,know as the loop body, until the given condition is false or 0.

for Loop

The for loop of Python is designed to process the items of any sequence, such as a list or string, one by one.


for variable in sequence:
      #statements_to_repeat

Example

            
for i in range(1, 101, 1):
   print(i)

while Loop

A while loop is a conditional loop that will repeat the instructions within itself as long as a conditional remains true.

            
   while (logical-expression):
   #loop Body

Example:

            
 i = 1
while (i <= 100):
    print(i)
    i = i + 1

break statement

The break statement enables a program to skip over a part of the code. A break statement terminatees or stops the very loop it lies within and go the external body.

            
   for variable in sequence:
      #statement1
      if (condition):
         break
      #statement2
   statements_after_loop

Example:

            
for i in range(1, 101, 1):
   print(i, end=" ")
   if (i == 50):
      break
   else:
      print("banana")
print("Thank you")

continue statement

The continue statement skips the rest of the loop statements and causes the next iteration to occur.

            
for variable in sequence:
      #statement1
      if :
         continue
         #statement2
         #statement3
         #statement4

Example:

            
for i in [2, 3, 4, 6, 8, 0]:
   if (i % 2 != 0):
      continue
   print(i)

1.Strings

Python string is a sequence of characters, and each character can be individually accessed using its index.

Strings

You can create strings by enclosing text in both forms of quotes - single quotes or double quotes:

            
variable_name = "String Data" #syntax
str = "CodeWithIshant"
print("String is", str)

Indexing

The position of every character placed in the string starts from the 0th position and step by step it ends at length-1 position.

Slicing refers to obtaining a sub-string from the given string. The following code will include index 2, 3, and 4 for the variable named variable_name.

            
string_var[integer_start_value:integer_stop_value:integer_step_value]  #syntax
variable_name[2:5]

Here start and step value are considered 0 and 1 respectively if not mentioned/given/provided by the programmer.

String Methods

Case Conversion

            
text = "  Hello, Ishant!  "
print(text.lower()) # lower() → " hello, ishant! "
print(text.upper()) # upper() → " HELLO, ISHANT! "
print(text.title()) # title() → " Hello, Ishant! "
print(text.capitalize()) # capitalize() → " hello, ishant! "
print(text.swapcase()) # swapcase() → " hELLO, iSHANT! "

Whitespace and Padding

            
print(text.strip()) # strip() → "Hello, Ishant!"
print(text.lstrip()) # lstrip() → "Hello, Ishant! "
print(text.rstrip()) # rstrip() → " Hello, Ishant!"
print("42".zfill(5)) # zfill() → "00042"
print("hi".center(10)) # center() → " hi "
print("hi".ljust(10)) # ljust() → "hi "
print("hi".rjust(10)) # rjust() → " hi"

Searching and Count 
            
print(text.find("Ishant")) # find() → 9 (index of first match)
print(text.rfind("l")) # rfind() → 11 (last index of "l")
print(text.index("Hello")) # index() → 2 (raises error if not found)
print(text.count("l")) # count() → 3 (number of times "l" appears)

Start/End Checks

            
print(text.startswith(" He")) # startswith() → True
print(text.endswith("! ")) # endswith() → True

Replacing and Splitting

            
print(text.replace("Ishant", "Dev")) # replace() → " Hello, Dev! "
print(text.split(",")) # split() → [' Hello', ' Ishant! ']
print(" | ".join(["Ishant", "Python"])) # join() → "Ishant | Python"

Validation Methods

            
print("123".isdigit()) # isdigit() → True
print("abc".isalpha()) # isalpha() → True
print("abc123".isalnum()) # isalnum() → True
print("Hello".isupper()) # isupper() → False
print("hello".islower()) # islower() → True
print(" ".isspace()) # isspace() → True
print("Hello".istitle()) # istitle() → True

Encoding and Formatting

            
print("Python".encode()) # encode() → b'Python' (bytes)
print("Hello {0}".format("Ishant")) # format() → "Hello Ishant"
print(f"Hello {'Ishant'}") # f-string → "Hello Ishant"

Casefold for Aggresive Lowering

            
print("Straße".casefold())  # casefold() → "strasse"

Partitioning

            
print(text.partition(",")) # partition() → (' Hello', ',', ' Ishant! ')
print(text.rpartition(",")) # rpartition() → (' Hello', ',', ' Ishant! ')

Splitting by Lines

            
multiline = "Line1\nLine2\nLine3"
print(multiline.splitlines()) # splitlines() → ['Line1', 'Line2', 'Line3']

2.Lists

A list is python represents list of comma-separated values of any data type between square brackets


lst = [element1, element2, ............]

These elements can be of any data types

Indexing

the position of every element placed in the list starts from the 0th and the step by step it end at length 1 position. List is ordered, indexed, mutable and the most flexible and dynamic collection of elements in Python.

Empty list

this method allows you to create an empty list: 


empty_list = []

List Methods:

Adding Elements


fruits = ["apple", "banana", "mango"]
fruits.append("orange")     # append() → Adds 'orange' at the end
fruits.insert(1, "grape")   # insert() → Inserts 'grape' at index 1
fruits.extend(["kiwi", "pear"]) # extend() → Adds multiple items

Removing Elements


fruits.remove("banana")   # remove() → Removes first occurrence
fruits.pop()              # pop() → Removes and returns last item
fruits.pop(1)             # pop(index) → Removes at index 1
fruits.clear()            # clear() → Removes all items

Searching and Counting


fruits = ["apple", "banana", "apple", "grape"]
print(fruits.index("apple"))   # index() → First index of 'apple'
print(fruits.count("apple"))   # count() → Occurrences of 'apple'

Sorting and Reversing


fruits.sort()             # sort() → Alphabetical (in-place)
fruits.sort(reverse=True) # sort(reverse=True) → Descending
fruits.reverse()          # reverse() → Reverses the list

Sorted Copy (Without Modifying Original)


sorted_list = sorted(fruits)  # sorted() → Returns new sorted list

Copying and Joining


copy_list = fruits.copy()  # copy() → Shallow copy
combined = fruits + ["kiwi", "melon"]  # + → Concatenation

Looping and Membership


for fruit in fruits:
    print(fruit)           # Loop through list

print("apple" in fruits)    # in → Check existence
print("kiwi" not in fruits) # not in → Check non-existence

List Comprehension


squares = [x**2 for x in range(5)]        # → [0, 1, 4, 9, 16]
evens = [x for x in range(10) if x % 2 == 0] # → Even numbers

Nested Lists (2D Lists)


matrix = [[1, 2], [3, 4]]
print(matrix[0][1])   # → 2 (element at row 0, col 1)

3. Tuples

Tuples are represented as comma-separated values of any data type within parantheses unlike list. It's a ordered, indexed but non-mutable collection of comma-separated elements.

Indexing

The position of every element placed in the tuple starts from the 0th position and step by step it ends at length-1 position.

Creating Tuples


t1 = (1, 2, 3)         # Regular tuple
t2 = 1, 2, 3           # Tuple without parentheses
t3 = (5,)              # Single-element tuple (note the comma)
empty = ()             # Empty tuple

Accessing Tuple Elements


print(t1[0])           # → 1 (first element)
print(t1[-1])          # → 3 (last element)

Tuple Unpacking


a, b, c = t1
print(a)  # → 1
print(b)  # → 2
print(c)  # → 3

Tuple Slicing


print(t1[1:])          # → (2, 3)
print(t1[:2])          # → (1, 2)

Tuple Immutability


# t1[0] = 10 → Error: 'tuple' object does not support item assignment
# Tuples are immutable

Using Tuples in Loops


for item in t1:
    print(item)

Tuple Methods:


print(t1.count(2))     # → 1 (number of times 2 appears)
print(t1.index(3))     # → 2 (index of element 3)

Nesting and Combining Tuples


nested = (t1, t2)
combined = t1 + t2     # → (1, 2, 3, 1, 2, 3)

Tuple with Different Data Types


mix = ("Ishant", 18, 3.14, True)
print(mix[0])  # → "Ishant"

4. Sets

A set is a collection of multiple values in curly brackets. It is both unindexed and unordered.

Creating Sets


s1 = {1, 2, 3}           # Set with integers
s2 = set([4, 5, 6])      # Set from list
s3 = set("hello")        # Set from string (no duplicates)
empty_set = set()        # Correct way to create empty set

As Set is an unorederd, immutable, non-indexed type of collection. Duplicate elements are not allowed in it.

Adding Elements to Set


s1.add(4)                # Adds a single element
s1.update([5, 6])        # Adds multiple elements

Removing Elements from Set


s1.remove(2)             # Removes element; raises error if not present
s1.discard(10)           # Removes element; does nothing if not present
s1.pop()                 # Removes and returns an arbitrary element
s1.clear()               # Empties the set

Set Operations (Union, Intersection, etc.)


a = {1, 2, 3}
b = {3, 4, 5}

print(a | b)             # Union → {1, 2, 3, 4, 5}
print(a & b)             # Intersection → {3}
print(a - b)             # Difference → {1, 2}
print(a ^ b)             # Symmetric difference → {1, 2, 4, 5}

Set Membership and Looping


print(2 in a)            # → True
for item in b:
    print(item)

Set Comparisons


a = {1, 2}
b = {1, 2, 3}

print(a.issubset(b))     # → True
print(b.issuperset(a))   # → True
print(a.isdisjoint({4})) # → True (no common elements)

Frozen Sets (Immutable Sets)


f = frozenset([1, 2, 3])
# f.add(4) → Error: 'frozenset' object has no attribute 'add'

5. Dictionaries

The dictionary is an unordered set off comma-separated key:value within curly brackets. It is required that in dictioanry no duplicate keys are allowed , values are allowed.

Creating Dictionaries


person = {"name": "Alice", "age": 25}
data = dict(city="Delhi", pin=110001)
empty = {}               # Empty dictionary

Accessing Values


print(person["name"])     # → Alice
print(person.get("age"))  # → 25
print(person.get("gender", "Not Found"))  # → Default value if key doesn't exist

Modifying Dictionaries


person["age"] = 26        # Update value
person["gender"] = "F"    # Add new key-value pair

Removing Items


del person["age"]         # Removes key 'age'
person.pop("gender")      # Removes and returns value of 'gender'
person.clear()            # Empties the dictionary

Looping Through Dictionary


for key in person:
    print(key, person[key])   # Loop through keys

for k, v in person.items():
    print(f"{k}: {v}")        # Loop through key-value pairs

Dictionary Methods


keys = person.keys()          # Returns all keys
values = person.values()      # Returns all values
items = person.items()        # Returns key-value pairs
copy_person = person.copy()   # Shallow copy

Using Dictionaries with `in`


print("name" in person)       # → True
print("salary" not in person) # → True

Nested Dictionaries


students = {
    "A": {"marks": 85, "age": 17},
    "B": {"marks": 90, "age": 18}
}
print(students["A"]["marks"])  # → 85

File Handling

File Handling refers o reading or writing data form different types of files.

open() Function


variable_name = open("file name","mode")

Modes

  1. r - to read content from file.
  2. w - to write the content into file.
  3. a - to append the existing content into file.
  4. r+ - to read and write into the file. The previous data in the file will be cleared/deleted.
  5. w+ - to write and read data . It will override the existing data.
  6. a+ - to append and read data from the file. It won't override the existing data.

Above all are for txt or csv file

Just add "b" after the first letter then it will be same case but for binary file.
